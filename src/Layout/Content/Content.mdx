# Content Component Documentation

The **Content** component provides a flexible container that adds spacing around its content. It accepts a `pad` prop which can be a boolean (to enable default padding) or a number (to specify a custom padding value).

## Key Props
- **pad:** A boolean or number that determines the padding applied to the component.
