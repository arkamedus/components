# Divider

The Divider component is used to visually separate content within a layout. It supports two orientations:

- **Horizontal** (default): A horizontal line which can be used to separate sections of content.
- **Vertical**: A vertical line that can be used to delineate content in a horizontal flow.

This component adheres to the oakd design system with a utility class structure for flexible and maintainable styling.

## Props

- **orientation** (optional): Determines the orientation of the divider. Accepts "horizontal" or "vertical". Defaults to "horizontal".
- **className** (optional): Additional class names for custom styling.
