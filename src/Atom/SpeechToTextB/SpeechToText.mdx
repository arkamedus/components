# SpeechToText Component

The `SpeechToText` component allows users to convert speech to text using the Web Speech API. It provides a button to start/stop voice recognition and displays the transcribed text.

## Props

- **buttonText** (string): The text displayed on the button. Defaults to "Start Speaking".
- **onChange** (function): Callback function that is called with the transcribed text when it changes.
- **placeholder** (string): Placeholder text to guide users on what to say.
- **title** (string): Optional title to display above the component.
