# AudioInput Component

The `AudioInput` component is designed to enable audio recording functionality within a web application. It leverages the MediaRecorder API to capture audio from the user's microphone and provides a simple interface for playback and reset of the recorded audio.

## Key Features
- **Audio Recording**: Users can record audio by clicking the microphone button.
- **Playback**: Recorded audio can be played back using the built-in audio controls.
- **Reset**: Users can reset the audio input, clearing the current recording.
- **Disabled State**: The component can be disabled to prevent user interaction.
