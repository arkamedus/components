# Collapsible

The `Collapsible` component is designed to hide or show content based on user interaction. It provides a toggleable section that can expand or collapse, making it ideal for displaying additional information without overwhelming the user interface.

# Collapsible Component

## Key Features
- **Toggle Functionality**: Users can expand or collapse the section to view or hide content.
- **Animation**: Smooth transitions for height and opacity when toggling.
- **Custom Action Area**: Supports an action area (e.g., buttons) alongside the title for contextual controls.
