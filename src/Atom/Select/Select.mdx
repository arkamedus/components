# Select Component

The Select component allows users to select an option from a dropdown list. It accepts an array of options, supports optional categorization, and renders custom placeholder text via the `placeholder` prop. The component also supports a `size` prop to adjust the visual size of the trigger button, ensuring consistency within oakd’s design system.
